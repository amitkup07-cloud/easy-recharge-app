# ⚡ Easy Recharge App

A complete utility and recharge application built with **Flutter** and **Firebase**.

## 🚀 Key Features
* **Mobile & DTH Recharge:** Dynamic UI for selecting various operators and plans.
* **Secure Transactions:** Safe and reliable payment integration flows.
* **Real-Time Updates:** Instant recharge confirmation using Firebase database.
* **User Authentication:** Secure login, password management, and profile handling.
